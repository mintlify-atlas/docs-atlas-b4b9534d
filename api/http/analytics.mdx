<Note>

</Note>

Analytics endpoints provide insights into learning progress, velocity, activity patterns, and concept-specific timelines.

## Concept Timeline

Retrieve mastery progression over time for a specific concept.

```bash GET /api/analytics/timeline/:conceptId
curl https://api.entendi.dev/api/analytics/timeline/python-list-comprehensions \
  -H "x-api-key: your_api_key_here"
```

### Response

```json
{
  "conceptId": "python-list-comprehensions",
  "currentMastery": {
    "min": 68,
    "max": 88,
    "display": "78%"
  },
  "timeline": [
    {
      "eventId": 12340,
      "timestamp": "2026-02-15T10:00:00.000Z",
      "mastery": { "min": 35, "max": 65, "display": "50%" },
      "eventType": "probe_response",
      "rubricScore": 2,
      "integrityScore": 0.92
    },
    {
      "eventId": 12345,
      "timestamp": "2026-03-01T14:30:00.000Z",
      "mastery": { "min": 68, "max": 88, "display": "78%" },
      "eventType": "probe_response",
      "rubricScore": 3,
      "integrityScore": 0.95
    }
  ]
}
```

<ResponseField name="conceptId" type="string" required>
  Concept identifier
</ResponseField>

<ResponseField name="currentMastery" type="object">
  Current mastery range (null if never assessed)
</ResponseField>

<ResponseField name="currentMastery.min" type="number" required>
  Lower bound of mastery confidence interval (0-100)
</ResponseField>

<ResponseField name="currentMastery.max" type="number" required>
  Upper bound of mastery confidence interval (0-100)
</ResponseField>

<ResponseField name="currentMastery.display" type="string" required>
  Display string (e.g., "78%" or "68-88%" if uncertain)
</ResponseField>

<ResponseField name="timeline" type="array" required>
  Assessment events ordered by timestamp (up to 200 events)
</ResponseField>

<ResponseField name="timeline[].eventId" type="number" required>
  Assessment event ID
</ResponseField>

<ResponseField name="timeline[].timestamp" type="string" required>
  ISO 8601 timestamp
</ResponseField>

<ResponseField name="timeline[].mastery" type="object" required>
  Mastery range at this point in time
</ResponseField>

<ResponseField name="timeline[].eventType" type="string" required>
  Assessment type
</ResponseField>

<ResponseField name="timeline[].rubricScore" type="number" required>
  Rubric score (0-4)
</ResponseField>

<ResponseField name="timeline[].integrityScore" type="number">
  Integrity score (0-1)
</ResponseField>

## Aggregate Timeline

Retrieve learning progress across all concepts, grouped by day.

```bash GET /api/analytics/timeline
curl https://api.entendi.dev/api/analytics/timeline \
  -H "x-api-key: your_api_key_here"
```

### Response

```json
{
  "timeline": [
    {
      "date": "2026-02-15",
      "assessments": 8,
      "avgDelta": 0.05,
      "cumulativeDelta": 0.05
    },
    {
      "date": "2026-02-16",
      "assessments": 12,
      "avgDelta": 0.08,
      "cumulativeDelta": 0.13
    }
  ]
}
```

<ResponseField name="timeline" type="array" required>
  Daily learning progress points (up to 500 recent events)
</ResponseField>

<ResponseField name="timeline[].date" type="string" required>
  Date in YYYY-MM-DD format
</ResponseField>

<ResponseField name="timeline[].assessments" type="number" required>
  Number of assessments on this day
</ResponseField>

<ResponseField name="timeline[].avgDelta" type="number" required>
  Average mastery gain per assessment (as probability delta, 0-1 scale)
</ResponseField>

<ResponseField name="timeline[].cumulativeDelta" type="number" required>
  Cumulative mastery gain from start to this day
</ResponseField>

## Learning Velocity

Retrieve learning velocity metrics over rolling time windows (7 days, 30 days, 90 days).

```bash GET /api/analytics/velocity
curl https://api.entendi.dev/api/analytics/velocity \
  -H "x-api-key: your_api_key_here"
```

### Response

```json
{
  "velocity": {
    "7d": {
      "assessments": 25,
      "avgDelta": 0.06,
      "conceptsAssessed": 12
    },
    "30d": {
      "assessments": 98,
      "avgDelta": 0.05,
      "conceptsAssessed": 32
    },
    "90d": {
      "assessments": 245,
      "avgDelta": 0.04,
      "conceptsAssessed": 67
    }
  },
  "snapshots": [
    {
      "userId": "usr_abc123",
      "date": "2026-02-15",
      "assessmentCount": 8,
      "conceptsAssessed": 6,
      "avgMasteryDelta": 0.05,
      "totalDismissals": 1,
      "avgIntegrityScore": 0.92,
      "probeCount": 8,
      "tutorCount": 2,
      "domains": { "programming": 5, "mathematics": 3 }
    }
  ]

```

<ResponseField name="velocity" type="object" required>
  Velocity metrics for rolling time windows
</ResponseField>

<ResponseField name="velocity.7d" type="object" required>
  Last 7 days
</ResponseField>

<ResponseField name="velocity.7d.assessments" type="number" required>
  Total assessments in window
</ResponseField>

<ResponseField name="velocity.7d.avgDelta" type="number" required>
  Average mastery gain per assessment
</ResponseField>

<ResponseField name="velocity.7d.conceptsAssessed" type="number" required>
  Number of unique concepts assessed
</ResponseField>

<ResponseField name="snapshots" type="array" required>
  Daily snapshot data (up to 90 days, ordered by date ascending)
</ResponseField>

<ResponseField name="snapshots[].date" type="string" required>
  Date in YYYY-MM-DD format
</ResponseField>

<ResponseField name="snapshots[].assessmentCount" type="number" required>
  Assessments on this day
</ResponseField>

<ResponseField name="snapshots[].conceptsAssessed" type="number" required>
  Unique concepts assessed
</ResponseField>

<ResponseField name="snapshots[].avgIntegrityScore" type="number">
  Average integrity score (0-1)
</ResponseField>

<ResponseField name="snapshots[].domains" type="object" required>
  Assessment count by domain (e.g., `{"programming": 5}`)
</ResponseField>

## Activity Heatmap

Retrieve daily assessment counts for calendar heatmap visualization.

```bash GET /api/analytics/activity-heatmap
curl https://api.entendi.dev/api/analytics/activity-heatmap?days=365 \
  -H "x-api-key: your_api_key_here"
```

### Query Parameters

<ParamField query="days" type="number" default="365">
  Number of days to include (lookback period)
</ParamField>

### Response

```json
{
  "heatmap": [
    {
      "date": "2025-03-02",
      "assessmentCount": 8,
      "conceptsAssessed": 6,
      "avgMasteryDelta": 0.05,
      "domains": { "programming": 5, "mathematics": 3 }
    },
    {
      "date": "2025-03-03",
      "assessmentCount": 12,
      "conceptsAssessed": 9,
      "avgMasteryDelta": 0.07,
      "domains": { "programming": 8, "science": 4 }
    }
  ]
}
```

<ResponseField name="heatmap" type="array" required>
  Daily activity snapshots ordered by date ascending
</ResponseField>

<ResponseField name="heatmap[].date" type="string" required>
  Date in YYYY-MM-DD format
</ResponseField>

<ResponseField name="heatmap[].assessmentCount" type="number" required>
  Total assessments on this day
</ResponseField>

<ResponseField name="heatmap[].conceptsAssessed" type="number" required>
  Unique concepts assessed
</ResponseField>

<ResponseField name="heatmap[].avgMasteryDelta" type="number" required>
  Average mastery gain
</ResponseField>

<ResponseField name="heatmap[].domains" type="object" required>
  Assessment count per domain
</ResponseField>
